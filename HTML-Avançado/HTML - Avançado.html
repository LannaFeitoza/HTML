Criando Tabelas

● Criando tabelas

A tag para criar tabelas: < table >< /table >
Além da tag < table > são necessárias mais duas tags:

  • < td >< /td > - Cria as células
  • < tr > - Cria as linhas

Exemplo:

Nome Sobrenome
Lanna Feitoza
Luana Fernandes
Janisson Silva 

● Sintaxe do exemplo: 
            < table >
                < tr >
                    < td >Nome< /td >
                    < td >Sobrenome< /td >
                < tr >
                    < td >Lanna< /td >
                    < td >Feitoza< /td >
                < tr >
                    < td >Luana< /td >
                    < td >Fernandes< /td >
                < tr >
                    < td >Janisson< /td >
                    < td >Silva< /td >
            < /table >

● Adicionando bordas na tabela

Para adicionar bordas é usado o atributo border, a espessura é definida em pixels.
● Sintaxe: < table border="número de pixels" >

Exemplo:

Nome Sobrenome
Lanna Feitoza
Luana Fernandes
Janisson Silva

● Adicionando cor na borda

Para adicionar cor a borda é utilizado o atributo bordercolor.
● Sintaxe: < table bordercolor="cor" >

Exemplo:

Nome Sobrenome
Lanna Feitoza
Luana Fernandes
Janisson Silva

● Adicionando cor de fundo na tabela

Para adicionar cor de fundo na tabela é utilizado o atributo bgcolor.
● Sintaxe: < table bgcolor="cor" >

Exemplo:

Nome Sobrenome
Lanna Feitoza
Luana Fernandes
Janisson Silva

Também pode adicionar cor apenas em uma linha:
● Sintaxe: < tr bgcolor="cor" >

Exemplo:

Nome Sobrenome
Lanna Feitoza
Luana Fernandes
Janisson Silva

Também pode adicionar cor em apenas uma célula.
● Sintaxe: < td bgcolor="cor" >

Exemplo:

Nome Sobrenome
Lanna Feitoza
Luana Fernandes
Janisson Silva

● Adicionando imagem de fundo na tabela

Para adicionar uma imagem no fundo da tabela é utilizado o atributo background.
● Sintaxe: < table background="nome da imagem.extensão" >

Exemplo:

Nome Sobrenome
Lanna Feitoza
Luana Fernandes
Janisson Silva

● Adicionando largura e altura

Para adicionar largura e altura são usados os atributos width e height.
Os valores podem ser inseridos em pixels ou porcentagem.
●Sintaxe: < table width="valor" heigth="valor" >

Exemplo:

Nome Sobrenome
Lanna Feitoza
Luana Fernandes
Janisson Silva

● Adicionando alinhamento na tabela

Para fazer o alinhamento da tabela em relação à página web é usado o atributo align.
● Sintaxe: < table align="alinhamneto" >

  • Align="left" alinha na esquerda.
  • Align="center" alinha no centro.
  • Align="right" alinha na direita.

Exemplo de alinhamento no centro:

Nome Sobrenome
Lanna Feitoza
Luana Fernandes
Janisson Silva

● Adicionando espaço entre borda e célula

Para adicionar espaço entre a borda da tabela e as células é usado o atributo cellspacing.
● Sintaxe: < table cellspacing="número" >

Exemplo:

Nome Sobrenome
Lanna Feitoza
Luana Fernandes
Janisson Silva

● Atributos da tag TD

  1. Align: Permite alinhar horizontalmente o texto na célula.
    Sintaxe: < td aling="alinhamento" >
    • Align="left" - alinha na esquerda.
    • Align="center" - alinha no centro.
    • Align="right" - alinha na direita.

  2. Valing: Permite alinhar verticalmente o texto na célula.
    Sintaxe: < td valign="alinhamneto" >
    • Valign="top" - alinha na parte superior.
    • Valign="bottom" - alinha na parte inferior.
    • Valign="center" - alinha o centro.

  3. Colspan: Permite mesclar colunas.
    Sintaxe: < td colspan="número da quantidade de colunas" >

  4. Rowspan: Permite mesclar células.
    Sintaxe: < td rowspan="número da quantidade de células" >

Exemplo do atributo aling="right" na posição Feitoza.

Nome Sobrenome
Lanna Feitoza
Luana Fernandes

Exemplo do atributo valing="bottom" na posição Feitoza.

Nome Sobrenome
Lanna Feitoza
Luana Fernandes

Exemplo do atributo colspsan="2" para unir o Nome com mais 1 coluna.

Nome
Lanna Feitoza
Luana Fernandes

Exemplo do atributo rowspan="2" para unir o Feitoza com mais uma célula.

Nome Sobrenome
