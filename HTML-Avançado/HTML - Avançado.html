● Criando tabelas

A tag para criar tabelas: < table >< /table >
Além da tag < table > são necessárias mais duas tags:

  • < td >< /td > - Cria as células
  • < tr > - Cria as linhas

Exemplo:

Nome Sobrenome
Lanna Feitoza
Luana Fernandes
Janisson Silva 

● Sintaxe do exemplo: 
            < table >
                < tr >
                    < td >Nome< /td >
                    < td >Sobrenome< /td >
                < tr >
                    < td >Lanna< /td >
                    < td >Feitoza< /td >
                < tr >
                    < td >Luana< /td >
                    < td >Fernandes< /td >
                < tr >
                    < td >Janisson< /td >
                    < td >Silva< /td >
            < /table >

● Atributos da tag TD

  1. Align: Permite alinhar horizontalmente o texto na célula.
    Sintaxe: < td aling="alinhamento" >
    • Align="left" - alinha na esquerda.
    • Align="center" - alinha no centro.
    • Align="right" - alinha na direita.

  2. Valing: Permite alinhar verticalmente o texto na célula.
    Sintaxe: < td valign="alinhamneto" >
    • Valign="top" - alinha na parte superior.
    • Valign="bottom" - alinha na parte inferior.
    • Valign="center" - alinha o centro.

  3. Colspan: Permite mesclar colunas.
    Sintaxe: < td colspan="número da quantidade de colunas" >

  4. Rowspan: Permite mesclar células.
    Sintaxe: < td rowspan="número da quantidade de células" >

● Criando Formulários

Para criar um formulário é usada a tag form.
● Sintaxe: < form atributos > < /form >

Para mandar os dados inseridos em um formulário é usada o atributo action.
Esse atributo especifica a URL do script que processará os dados do formulário.
● Sintaxe: < form action = "URL" >

Exemplo: < form action="mailto:alguem@gmail.com" >

Para definir a forma como serão mandados os dados temos o atributo method.
Ele pode receber dois valores: get ou post.
O post consegue enviar mais dados para o servidor enquanto o get tem um menor envio de dados.
● Sintaxe: < form action ="URL" method="get" >

Também temos o atributo type, este atributo que vai definir o tipo de entrada,
pode ser uma entrada de texto, de senha, única escolha, mútipla escolha.

E para ler a entrada temos o atributo input, é a partir dele que temos a entrada.

● Criando um campo de texto

● Sintaxe: < input=text name="nome" size="número" maxlenght="número" >

  1. Type: valor que especifica o tipo de entrada. Nesse caso tipo text.
  2. Name: nome da vriável que armazena os dados.
  3. Size: tamanho do campo em número de caracteres.
  4. Maxlenght: número máximo de caracteres que podem ser inseridos pelo usuário.

Exemplo: 

Dentro do HTML: 
            < form action="não definido" method="get" >
                < input type="text" name="nome" size="30" maxlenght="50" >
            < /form >

Para mostrar um padrão de como dever inserir o dado, temos o atributo value.
● Sintaxe: < input type=text name="nome" size="número" value="Ex: AM" >

● Criando opção em lista e menu

Para criar uma opção em lista é usado a tag select, com o atributo multiple. Junto com a tag option.

● Sintaxe:  < form >
                < select multiple name="identifica a lista" >
                    < option value="1" >Nome do item
                < /select> 
            < /form >

Exemplo:

Para criar um menu é só tirar o atributo multiple da tag select.

● Sintaxe:  < form >
                < select name="identifica a lista" >
                    < option value="1" >Nome do item
                < /select> 
            < /form >

Exemplo:

Para que no menu apareça um valor padrão é só utilizar o atributo selected na tag option.
● Sintaxe: < option selected value="1" >

Sobre os atributos:

  1. Multiple: exibe a lista aberta.
  2. Name: identifica a lista.
  3. Option: cria cada item da lista.
  4. Selected: mostra o valor padrão da lista.
  5. Value: valor que será retornado.
