Estilos de Fonte + quebra de linha

  • Para quebrar linha no HTML: < br >

  • Para deixar em negrito: < b >< / b >
  • Para deixar em itálico: < i >< / i >
  • Para deixar sublinhado: < u >< / u >

Alinhamento de Texto

● Para deixar o texto alinhado a esquerda: p align=left

● Para deixar o texto alinhado no centro: p align=center

● Para deixar o texto alinhado a direita: p align=right

Mudar Fonte, Tamanho e Cor do Texto

  • Fonte: < font face="nome da fonte" >
  • Tamanho: < font size=" 1 <= x <= 7" >
  • Cor: < font color="nome ou número da cor">

Linha horizontal

● Para por linha horizontal: < hr >

    ● Para personalizar a lista ordenada: < ol type= "estilo do número" >

    1. A- Para exibir letras maiúsculas
    2. a- Para exibir letras minúsculas
    3. I- Para exibir números romanos grandes
    4. i- Para exibir números romanos pequenos

  2. Lista Não Ordenada: < ul > < li > < /ul >

    • Macacos podem escalar
    • Grilos podem saltar
    • Cavalos podem correr
    • Corujas podem caçar
    • Guepardos podem correr
    • Águias podem voar
    • Pessoas podem tentar
    • Mas isso é tudo.

    ● Para personalizar a lista não ordenada: < ul type= "estilo do marcador" >

    • disc- Para exibir ●
    • circle- Para exibir ○
    • square- Para exibir ▪

  3. Lista Intercalada: < ul > < li > < ol > < li > < /ol > < /ul >

    • Parte UL 1
      1. Parte OL 1
      2. Parte OL 1

    • Parte UL 2
      1. Parte OL 2
      2. Parte OL 2

    ● Ou pode ser o oposto: < ol > < li > < ul > < li > < /ul > < /ol >

    1. Parte OL 1
      • Parte UL 1
      • Parte UL 1

    2. Parte OL 2
      • Parte UL 2
      • Parte UL 2

Inserir Imagens

● Sintaxe: < img srg="C:\pasta\nome imagem.extensão da imagem" >
● Para alinhar a imagem: < img src="nome da imagem" align= "tipo alinhamento" >

  • TOP - Para deixar a imagem no topo do texto.
  • BOTTOM - Para deixar a imagem abaixo do texto.
  • MIDDLE - Para deixar a imagem no meio do texto.
  • RIGHT - Para deixar imagem a direita do texto.
  • LEFT - Para deixar imagem a esquerda do texto.

● Para aparecer mensagem ao passar cursor na imagem: < img alt="Texto Explicativo" >

Criar Links

  • Criar links na página web:

    Os links podem ser criados no texto e também em outros elementos do website, como imagens ou itens de um menu. Será usada a tag ANCHOR.
    ● Sintaxe: < a > < /a >
    ● Sintaxe inserindo o link: < a href="URL" > Texto descritivo < /a >

    Clique aqui para acessar o complemento de teste deste html.

  • Criar links para uma seção:

    Para criar links para uma seção é necessário usar a tag: < a name="nomeação da seção" > para dar um nome a sua seção ou usar o prório nome. Depois é só usar a tag: < a href="#nomeação da seção" > Texto Descritivo < /a >.

  • Criar links em uma imagem:

    Para criar links em uma imagem: < a href="URL" > < img src="img_url" > < /a >

  • Personalizar a cor dos links:

    São três elementos:

    1. link: Especifica a cor dos links não acessados da página web.
    2. vlink: Após ser acessado, o link apresentará a cor especificada por este atributo.
    3. alink: Especifica a cor que o linl deverá apresentar quando o cursor estiver sobre ele.

    ● Sintaxe: < body link="cor" vlink="#cor" alink="cor" >

    Entretanto, essa sintaxe vai mudar a cor de todos os links para terem o mesmo padrão. Mas se quiser deixar cada link um de cada cor, basta usar a sintaxe do Font Color.

    ● Sintaxe: < a href="URL" >< font color="cor" > Texto Descritivo < /font >< /a >

  • Criar links em lista

    Para fazer isso deve inserir a tag ANCHOR ao lado do < li > na lista. Se for link de uma de seção é só criar depois as tags name de acordo com sua seção.
